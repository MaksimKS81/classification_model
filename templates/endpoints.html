    • ./process_json_decim

    Description of the /process_json_decim (Decimated tested and reference time-series) endpoint.

    Parameters:

    • mode: Server mode: 'link' or 'data'.
    • test_file_name: If server works in 'link' mode we use link to local file.
    • data: If server works in 'data' mode we read json file on client's side and send it as a paramiter.

    Answer:

    • Processing time: Processing time in sec.
    • Label: Record's label recived from model .
    • N_CPU: Number of CPU cores used for processing.
    • test_file_name: For 'link'mode it will be local file name in 'data' mode will be 'data_mode'.
